Overview

Purpose

The purpose of this site is to display information from a SQL database in a front-end format. It will display information about dinosaurs.

Audience

The target audience for this will be people who like dinosaurs and want to learn more information about them.

This site will make use of a search system that will allow the user to grab data.

Content

Home Page

Introduction to the website. A contact form at the bottom to reach me. Images will be taken from Wikipedia with a link embedded taking the user directly to the wiki page.

All Dinosaurs

This will feature a list of every dinosaur in the database with a search bar at the top and a filter form that will allow searching by tags within the database. A picture for each dino of their bones will be displayed alongside the brief introduction to the dinosaur.

Wireframes

Home

The home page features a welcome message, a button that will take the users directly to the second page, a large image of a T-Rex skeleton, three separate images that act as links to the tags for time periods, and a form that will allow the user to contact me if needed.

Dinosaur Archive

The second page acts as the display for all the dinosaurs in the SQL database. There is a search bar at the top and a filter that will allow the user to filter the results they want.
